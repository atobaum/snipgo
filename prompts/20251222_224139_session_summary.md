## 1. Git 파일 추적 문제 해결

### 문제
- `cmd/snipgo/config.go` 파일이 git에 포함되지 않음
- `.gitignore`의 `snipgo` 패턴이 `cmd/snipgo/` 디렉토리까지 무시하고 있었음

### 해결
1. **`.gitignore` 수정**
   - `snipgo` → `/snipgo`로 변경
   - 루트의 `snipgo` 바이너리만 무시하도록 수정

2. **파일 추가 및 정리**
   - `cmd/snipgo/config.go`를 git에 추가
   - 루트의 잘못된 `config.go` 파일 삭제

### 수정된 파일
- `.gitignore` - 패턴 수정 (`snipgo` → `/snipgo`)
